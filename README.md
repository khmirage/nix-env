## System Configure 

### This project is **bash, vim, git** setting for Linux / Mac 

#### [Bash]
```
	bash_profile - for mac
	bashrc - for linux

	Symlink to $HOME
		bash_profile -> ~/.bash_profile
		bashrc -> ~/.bashrc
```

#### [Vim]
```
	vimrc - vim setting file
	vim - vundle submodule

	1. vundle submodule pull
		$ git submodule init
		$ git submodule update
	2. Symlink to $HOME
		vim -> ~/.vim
		vimrc -> ~/.vimrc
```

#### [Git]
```
	git global config

	Symlink to $HOME
		gitconfig -> ~/.gitconfig
```
